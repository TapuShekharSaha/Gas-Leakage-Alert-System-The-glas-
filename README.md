# Gas-Leakage-Alert-System-The-glas-

Overview

The Gas Leakage Alert System (GLAS) is a safety-focused IoT project designed to detect leakage of LPG gas in domestic environments. The system continuously monitors gas concentration levels and provides immediate alerts to prevent accidents and hazards.

When gas leakage is detected, the system triggers an alarm and sends a notification message to the user’s mobile device, enabling quick response and ensuring safety.

Objective

The primary objectives of this system are:

To detect LPG gas leakage in real time
To alert users instantly through an alarm and mobile notification
To minimize the risk of fire accidents and health hazards
Features
Real-time gas leakage detection
Automatic alarm activation using a buzzer
Mobile notification system (SMS or IoT-based alert)
Low-cost and efficient design
Suitable for home and small-scale applications
Technologies and Components Used
Gas Sensor (MQ-2 / MQ-5)
Microcontroller (Arduino / NodeMCU / Raspberry Pi)
Buzzer (Alarm System)
GSM Module or Wi-Fi Module (for sending notifications)
Power Supply
Working Principle
The gas sensor continuously monitors the concentration of LPG in the surrounding environment.
When the gas level exceeds a predefined threshold:
The microcontroller processes the sensor data
The buzzer is activated to alert nearby individuals
A notification message is sent to the user’s mobile device
The system continues monitoring until gas levels return to normal.
Applications
Residential homes
Kitchens and restaurants
LPG storage facilities
Industrial safety systems
Future Enhancements
Integration with a mobile application dashboard
Automatic gas valve shut-off mechanism
Cloud-based monitoring and logging
AI-based leakage prediction and analysis
Conclusion

The Gas Leakage Alert System (GLAS) provides a reliable and efficient solution for detecting LPG gas leaks and issuing timely alerts. It enhances safety and helps prevent potential accidents in both domestic and industrial environments.
